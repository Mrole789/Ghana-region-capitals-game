## Ghana-region-capitals-game
An interactive educational game to learn all 16 regions of Ghana and their capital cities, built with Python Turtle graphics. Inspired by the popular US states game on Sporcle!

## 🎮 About The Game  
Test your knowledge of Ghana's geography! This game helps you learn:

· All 16 regions of Ghana  
· Their capital cities  
· Their relative geographic locations on the map  

Accra is pre-placed as an example to get you started!

## 📁 Project Structure

```
ghana-regions-game/
├── main.py                          # Main game logic
├── ghana_map.gif                    # Ghana map image (not to scale) (Source)[https://www.mapsofindia.com/world-map/ghana/blank-outline-map.html]
├── 16_regions_and_capitals.csv      # Location data with coordinates
└── README.md                        # You are here!
```

## 🎯 How to Play

1. Run the game. 
2. Look at the map showing Ghana's regions  
3. Type any region or capital city name (except Accra, which is already shown)  
4. Correct answers appear on the map:  
   · 🏛️ Regions → Larger, bold font  
   · 🏙️ Capitals → Regular font  
5. Special commands:  
   · Type exit to end the game early  
   · Type nothing or wrong name → nothing happens  
6. Game ends when:  
   · You've named all 32 locations, OR
   · You type exit  
7. Missed locations are printed in the console at the end

## 🚀 Getting Started

Prerequisites

· Python 3.x  
· Required modules: turtle, pandas  

Installation

1. Clone the repository:  
   ```bash
   git clone https://github.com/yourusername/ghana-regions-game.git
   cd ghana-regions-game
   ```  
2. Install dependencies:  
   ```bash
   pip install pandas
   ```  
3. Add map image:  
   · Place ghana_map.gif in the project folder  
4. Run the game:  
   ```bash
   python main.py
   ```
